# http-server-go

## Instructions

To start this project

- Ensure you have Go installed
- Download / Clone the repo
- Navigate / cd into server folder
- Use `go run .`

## Testing

- Only three routes, `GET /ping`, `GET /hello`, `POST /echo`
- Use `curl` at your localhost endpoint at port 8080.
  - Example: `curl http://localhost:8080/ping`
